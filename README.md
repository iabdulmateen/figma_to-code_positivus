# 🚀 Positivus - Modern Digital Marketing Landing Page

A pixel-perfect, fully responsive landing page built with modern CSS techniques. This project showcases the implementation of complex layouts, interactive elements, and a clean UI based on the Positivus design concept.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom properties (variables), Flexbox, and CSS Grid for layout.
- **JavaScript:** For interactive elements like the accordion and mobile menu.

## ✨ Key Features
- **Fully Responsive:** Optimized for Desktop, Tablet, and Mobile screens.
- **Interactive UI:** Smooth hover effects on cards and buttons.
- **Working Accordion:** A clean "Working Process" section using JS.
- **Custom Slider:** Testimonials section with a unique design.
- **Contact Form:** Professional layout with custom-styled radio buttons.
